# DigitalClock

学部4年次の研究の一環として制作した  
**Android デジタル時計アプリ**です。

ユーザが時計を確認した回数に応じて，  
時計の表示形式が変化する仕組みを実装しています。

## 概要
- Android アプリとして動作するデジタル時計
- 時計の閲覧回数をカウント
- 条件に応じて表示形式を切り替え

## 構成
- `app/` : Android アプリ本体
- `gradle/`, `build.gradle.kts` など : ビルド・設定ファイル

※ サーバサイド処理は含まれていません。

## 開発環境
- Android Studio
- Kotlin
- Gradle
